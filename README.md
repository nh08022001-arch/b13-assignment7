#  KeenKeeper

## Project Overview

KeenKeeper is a modern web application designed to help users manage and nurture meaningful relationships. It allows you to track interactions with friends, monitor connection goals, and stay consistent with communication.

---

## Features

* **Friend Management**
  Add and view friends with detailed profiles including contact history, goals, and tags.

* **Interaction Tracking (Timeline)**
  Log interactions such as Call, Text, and Video with automatic date tracking and filtering.

* **Friendship Analytics**
  Visualize your communication habits using charts (Call / Text / Video breakdown).

---

##  Technologies Used

*  React
*  React Router
*  Tailwind CSS
*  DaisyUI
*  Recharts
*  React Hot Toast

---

##  Purpose

This project helps users stay connected with the people who matter most by providing a simple and intuitive system to track and maintain relationships.
